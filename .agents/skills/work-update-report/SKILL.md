# Work Update Report Skill

## Trigger
Use this skill when asked to generate a work update or latest work report.

## Instructions
- Summarize only the most recent work
- Keep it short and clear
- Focus on what changed and current status
- No code, no long explanations, no assumptions
- Return the exact format below, ready to copy

## Output Format

```
WORK UPDATE

DONE:
- ...

IN PROGRESS:
- ...

ISSUES:
- ...
```

## Fail Conditions
- Response is too long or unclear
- Does not reflect the latest work
- Wrong format
